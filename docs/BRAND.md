# ThaiPass — Brand Guide

## Name

**ThaiPass** — ваш пропуск в Таиланд. Премиум-сервис оформления виз, не массовое агентство.

- Domain target: `thaipass.ru`
- Tagline: «Визы в Таиланд без отказов и переделок»
- Subline: DTV · TR · Пенсионная 50+ · Москва и курьер по России

## Tone of voice

- Спокойный эксперт, не продавец «срочно-срочно»
- Конкретика: сроки, документы, риски
- Без emoji в интерфейсе
- Без слов: «вариант», «дешёво», «акция», «хит»

## Visual direction (phase 3)

- Palette: warm white `#FAFAF8`, deep teal `#0D4F4F`, sapphire `#1E3A5F`, gold accent `#C9A227`
- Fonts: Onest (UI) + Instrument Serif (headlines) — кириллица
- Motion: React Bits hero text, parallax background, Magic UI bento grid
- No stock tourist clichés (elephants, temples as wallpaper)
