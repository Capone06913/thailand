# Product Context

## Audience segments

| Segment | Pain | Package |
|---------|------|---------|
| Digital nomads / remote workers | DTV rejection, 500k THB docs, workcation proof | DTV Audit / Concierge |
| Urgent travelers | Flight in 3 days, no time for standard processing | TR Express |
| Planned tourists | Need TR with optimal price/time | TR Standard |
| Retirees 50+ | Want legal year-long stay, not visa runs | Retirement Path |

## Key fears (70% content)

- Consulate returned application due to wrong document format
- 500,000 THB on account but wrong statement seasoning
- Visa-free 60 days ending; possible reduction to 30 days in 2026
- Overstay fines and entry bans
- DTV vs TR confusion — wrong visa type chosen

## Trust signals (10%)

- Moscow office + courier
- Telegram channel with updated rules and prices
- Links to official Thai e-Visa and embassy
- Experience line (not long «our story» block)

## Competitor differentiation

Specialization **only Thailand visas** — not a tour agency, not real estate, not job placement.
