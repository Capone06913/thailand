# ThaiPass — финальная версия v2 (2026-06-10)

Снимок UI после полировки главной страницы.

## Что входит

- Hero: уменьшенная типографика, scroll-анимация без повторного появления текста
- Услуги: bento-сетка, **без цен** на главной, заголовок по центру
- Логотип: SVG-вордмарк (без PNG-артефактов)
- Процесс: уникальные анимации на каждой иконке
- Футер: компактный центрированный layout

## Git

- **Тег:** `final-v2`
- **Ветка:** `stable/final-v2`
- **Коммит:** см. `git show final-v2`

## Быстрый откат

Из корня репозитория:

```powershell
.\scripts\restore-final-v2.ps1
```

Или через git:

```powershell
git checkout final-v2
```

Чтобы вернуться к текущей разработке после просмотра:

```powershell
git checkout master
```
