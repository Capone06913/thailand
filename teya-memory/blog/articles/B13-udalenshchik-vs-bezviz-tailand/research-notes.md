# Research notes — B13 Удалёнщик vs безвиз

Дата: июнь 2026. Источники: fact-bank.md, thaievisa.go.th, moscow.thaiembassy.org.

## Ключевые факты

- **Безвиз** для граждан РФ — режим краткого туристического пребывания; правила и сроки меняются — проверять на moscow.thaiembassy.org.
- Работа удалённо на зарубежного работодателя **не формализована** в безвизовом режиме; юридически безвиз = туризм.
- **DTV Workcation** — долгосрочная виза с подтверждением удалённой работы, выпиской 500 000 бат, пакетом документов через thaievisa.go.th.
- **TR (туристическая)** — для визитов, не для легализации удалёнки на месяцы.
- TDAC обязателен при любом способе въезда.

## Сравнительная логика

| Критерий | Безвиз | TR | DTV Workcation |
|----------|--------|-----|----------------|
| Срок | Ограничен режимом | По визе | Долгосрочно |
| Удалёнка | Не оформлена | Нет | Да, с документами |
| 500k бат | Нет | Нет | Да |
