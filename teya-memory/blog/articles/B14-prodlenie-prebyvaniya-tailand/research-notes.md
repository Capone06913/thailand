# Research notes — B14 Продление пребывания

Дата: июнь 2026. Источники: fact-bank.md, thaievisa.go.th, moscow.thaiembassy.org.

## Ключевые факты

- Продление зависит от **типа въезда**: безвизовый stamp, туристическая TR, Non-Immigrant (DTV, O-A и др.).
- **Border run / visa run** — выезд и повторный въезд; правила и лимиты меняются; не универсальное «решение навсегда».
- Продление **туристического** пребывания — через иммиграционный офис (IMMI) по действующим правилам на дату обращения; размер сбора смотреть на официальных ресурсах.
- **DTV и O-A** продлеваются по своим категориям с отчётностью (90-day report для long-term).
- Перерасход срока без продления — штрафы и депортация; не рисковать «авось».
