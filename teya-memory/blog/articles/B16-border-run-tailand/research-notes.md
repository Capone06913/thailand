# Research notes — B16 Border run

Дата: июнь 2026. Источники: fact-bank.md, thaievisa.go.th, moscow.thaiembassy.org, immigration.go.th.

## Ключевые факты

- **Border run / visa run** — выезд и повторный въезд за новым stamp, не продление внутри страны.
- Иммиграция отслеживает **частые въезды**; возможны отказ, короткий stamp, дополнительные вопросы.
- **Overstay** не сбрасывается border run; штрафы в IMMI.
- Альтернативы: **продление IMMI** (TM.7), новая **TR**, **DTV** для удалёнщиков.
- **TDAC** на каждый новый въезд, отдельно от визы.

## Дифференциация

- B14: продление пребывания в IMMI.
- B16: риски border run и когда переходить на TR/DTV.
