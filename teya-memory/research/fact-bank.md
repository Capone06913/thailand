# Fact Bank — ThaiPass (июнь 2026)

Подтверждённые факты для статей. Не выходить за рамки без пометки «по данным на дату публикации».

## DTV (Destination Thailand Visa)

- Категории: Workcation (удалённая работа/фриланс), Soft Power (Muay Thai, тайская кухня, язык и др.).
- Требование по средствам: **500 000 THB** (или эквивалент в иной валюте по правилам консульства) — уточнять актуальные требования на thaievisa.go.th.
- Подача: через портал **thaievisa.go.th** (Thai e-Visa), как правило из-за пределов РФ — уточнять доступные консульства/ВAC для заявителей из РФ.
- DTV — **не** замена безвиза для краткого туризма; отдельный тип долгосрочного пребывания.
- ThaiPass **не гарантирует** одобрение — сопровождает подготовку документов.

## TDAC (Thailand Digital Arrival Card)

- Заменила бумажную TM6 для въезда.
- Заполнение: **tdac.immigration.go.th**
- Окно: **не ранее чем за 72 часа** до прибытия.
- TDAC — **не виза**, а карта прибытия для иммиграционного контроля.

## Безвиз (граждане РФ)

- На дату подготовки контента: безвизовый режим для туристических целей обсуждается/меняется; **не утверждать** финальные сроки без официального источника.
- Формулировка: «проверяйте актуальные правила на moscow.thaiembassy.org и thaievisa.go.th».
- При превышении срока безвиза или при цели «жить/работать» — нужна соответствующая виза (DTV, TR, O-A и т.д.).

## Туристическая виза TR

- Оформляется через консульский процесс / Thai e-Visa в зависимости от маршрута заявителя.
- ThaiPass TR Express / TR Standard — **услуги сопровождения**, не госпошлина.

## Пенсионная виза 50+ (O-A / Retirement Path)

- Возраст **50+**, финансовые требования по действующим правилам консульства.
- ThaiPass Retirement Path — сопровождение, не выдача визы государством.

## ThaiPass как сервис

- Офис: **Москва** (адрес — из site.inv / kontakty при публикации).
- Курьер по **России**.
- Заявки: форма на сайте → Telegram-уведомление оператору.
- **Нет** публичного телефона на сайте на старте (по project brief).

## Запрещено в контенте

- Выдуманные отзывы, рейтинги, кейсы с именами без источника.
- «100% одобрение», «гарантия визы».
- VPN/обход правил въезда.
- Точные суммы госпошлин без ссылки на официальный источник.
